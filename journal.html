Firebase Permission Setup

If you're seeing permission errors, you need to update your Firestore security rules:

  1. Go to the Firebase Console
  2. Select your project
  3. Click on "Firestore Database" in the left sidebar
  4. Go to the "Rules" tab
  5. Replace the existing rules with:
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}

Note: These rules allow anyone to read and write to your database. For a production app, you should implement proper authentication and security rules.
